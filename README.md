## ✨ Why ReactDashKit?

ReactDashKit is more than just a boilerplate—it's a **kickstart for modern dashboards** with powerful tools and clean code. Save days of setup and dive straight into building your product.

---

## 🧰 Features

- ⚛️ React + TypeScript + Redux Toolkit  
- 🎨 Material UI & MUI-X (data grid)  
- 🌙 Dark & Light theme toggle  
- 🛡️ Permission-based routing and rendering  
- 📦 Feature-wise folder structure & lazy loading  
- 💬 Axios integration for API calls  
- 📝 Formik + Yup for form handling and validation  
- 🧪 Jest for unit testing  
- 💅 SCSS support  
- 📱 Fully responsive layout  
- 🧹 Prettier integrated for clean code  
- 🌍 Environment-based config separation

## ⚙️ Getting Started

### 📁 1. Clone the Repository

```bash
git clone https://github.com/your-username/reactdashkit.git
cd reactdashkit
```

### 📦 2. Install Dependencies

```bash
npm install
```

### 🟢 3. Start the Development Server

```bash
npm start
```

Visit `http://localhost:3000` to see your app running.

---

## 🧪 Testing

### ▶️ Run All Tests

```bash
npm run test
```

### 👀 Run in Watch Mode

```bash
npm run test:watch
```

### 📊 Generate Test Coverage Report

```bash
npm run test:coverage
```

---

## 💅 Code Formatting

### 🧼 Format with Prettier

```bash
npm run format
```

This ensures your code stays clean and consistent.
